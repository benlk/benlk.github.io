Projects by Ben Keith

Projects on GitHub that I created, or heavily contributed to.

Lantern Timeline

Documentation and examples for students wishing to create an embeddable Timeline JS document for The Lantern.

Time

Going to bed? Use this project to set your alarm clock for the right wake-up time. Kinda like sleepyti.me. In need of improvement, because this sort of thing needn't be done in PHP.

Bookcase

A very lightweight Markdown→HTML conversion website platform, with templating in PHP. In progress.

Council for the Medieval and Renaissance Faire

From 2012-2014 I maintained the website for the Council for the Medieval and Renaissance Faire at The Ohio State University, including creating an easy way to switch the site between the different times of the year.

Project Largo

Project Largo is a responsively designed WordPress parent theme designed and optimized to support long form journalism. I'll be working on this during a Summer 2014 internship with its maintainer, the Investigative News Network.
